# 暖芽 · 评委演示脚本（3 分钟）

> 目标：一条丝滑主线，把"哇时刻"放最前和最后，让评委记住"技术的温度"。
> 建议分工：**主讲 1 人**（口播+操作）+ **救场 1 人**（盯网络/录屏/兜底）。

---

## 一、演示前 5 分钟检查清单（务必逐项验证！）

- [ ] 网络稳定，**手机热点已备用**（现场 WiFi 最不可靠）
- [ ] **传一张水瓶图给暖芽** → 回复要能说"我看到一个水瓶"（**不是**"左前臂瘀斑"）= 云函数真通
- [ ] **证据库传水瓶图** → 要说"未见明显人体伤情" = vision 真通
- [ ] 法律问答问一句 → 回答下方**有来源引用** = RAG 真通
- [ ] 生成书面陈述 → 能弹出正式陈述
- [ ] **录屏备份已录好**（完整跑通一遍录下来，翻车就放这个）
- [ ] 解锁密码 **1234** 记牢；准备好一张伤情示意图

> ⚠️ 最大风险：云函数没通时，传任何图都显示固定"左前臂瘀斑"——当场穿帮。**上台前一定用水瓶图验证！**

---

## 二、演示动线（逐步脚本 · 操作 + 口播 + 看点）

### ① 开场：一个"记账本" · 15s
- **操作**：打开小程序，是个普通记账本，随手记一笔账
- **口播**：「这看起来是个记账本小程序……但它其实是——」（停顿，制造悬念）
- **★看点**：反差铺垫，让评委疑惑

### ② 解锁：哇时刻 · 10s
- **操作**：在"金额"框输入 `1234`，点「记一笔」→ 界面瞬间变成真实首页
- **口播**：「输入密码，它变身成一个反家暴援助工具。为什么要伪装成记账本？因为家暴受害者最怕的，就是求助时被施暴者发现。」
- **★看点**：全场最强反差 + 点出核心洞察

### ③ 暖芽倾诉 + 危机检测 · 40s
- **操作**：进「倾诉」，发：`他拿刀威胁我，我活不下去了`
- **口播**：「受害者可以随时向 AI 倾诉，它始终相信她、不评判。注意——」（危机词触发顶部红色 Banner）「当 AI 检测到'拿刀''活不下去'这类信号，立刻弹出报警和求助通道。」
- **★看点**：AI 有温度 + 主动兜底安全（可以顺带指打字机流式、"💛暖芽正在认真想"的 Agent 状态）

### ④ 多模态看图 · 30s
- **操作**：在对话里发一张伤情照片
- **口播**：「她可以直接拍照，AI 会看图识别伤情、提醒尽快就医并保存证据。」（暖芽看图回应）
- **★看点**：多模态真看图（这里千万确保云函数通！）

### ⑤ 法律问答 · 真 RAG · 30s
- **操作**：进「法律问答」，问：`人身安全保护令怎么申请`
- **口播**：「这背后是 **569 篇真实法律法规和最高法典型案例**的知识库。看——」（指检索状态 🔍 → 回答下方来源引用）「每个回答都标注来源，不是 AI 瞎编。」
- **★看点**：真 RAG + 可信度（Agent 状态"检索 569 篇 → deepseek-v3 作答"很有过程感）

### ⑥ 证据整理 · 书面陈述 · 20s
- **操作**：进「我的记录」→「生成书面陈述」
- **口播**：「最后，AI 把零散的证据整理成一份可以直接用于报警、起诉的正式书面陈述。」
- **★看点**：从取证到用证的闭环价值

### ⑦ 快速离开：收尾 · 10s
- **操作**：点右下角绿色「快速离开」→ 秒回记账本
- **口播**：「如果施暴者突然出现——一键回到记账本，不留痕迹。」
- **★看点**：回到安全感，首尾呼应

### ⑧ 升华收尾 · 15s
- **口播**：「从隐蔽求助、AI 陪伴、法律支持，到证据固定——暖芽陪受害者走完求助的每一步。我们相信，技术的温度，是让最脆弱的人也能安全地伸出手。」

## 三、兜底应对（背下来，临场不慌）

| 情况 | 应对 |
|------|------|
| AI 回复慢/转圈 | 等 3 秒会自动出兜底回答，边等边讲设计理念，别冷场 |
| 某个 AI 功能彻底翻车 | 淡定跳过，口播"这块我们用录屏展示"，切录屏 |
| 网络断 | 救场同学立刻切手机热点 |
| 被问"为什么小程序不是 App" | 大方答："小程序零门槛、即用即走，最适合应急求助；隐蔽性上独立 App 更好，是我们的取舍" |
| 被问数据安全 | 讲架构：匿名身份、证据加密设计、Key 只在云端、一键清除（诚实说哪些是"已实现"、哪些是"设计方案"）|

4. 提前想好评委可能问的 3 个问题（技术实现 / 商业价值 / 安全合规）
